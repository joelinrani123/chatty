Chatty

Chatty is a simple real-time chat application built with React, Node.js, Stream, Twilio, and MongoDB. It allows users to send and receive messages instantly. Twilio is used for sending SMS notifications, and MongoDB is used to store user data.

//Technologies Used

Frontend:
React

Backend:
Node.js
Express.js

Database:
MongoDB

//APIs and Services:

Stream (for real-time chat functionality)

Twilio (for SMS notifications)
Features

User registration and login
Real-time messaging
Create and join chat channels
Store user data in MongoDB
Send SMS notifications using Twilio
Simple and responsive interface
